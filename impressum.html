Site Information

Details about the technical implementation and ownership of this website.

Technology Stack

This website is built with HTML5, CSS3, and Vanilla JavaScript. It follows a minimalist approach, avoiding heavy frameworks to ensure fast loading times and accessibility.

Design Inspiration

The visual language is inspired by Mediterranean landscapes and the symbolism of the olive tree (Ovilli). It uses a "glassmorphism" style to blend modern UI trends with organic colors.

Ownership

This is a personal portfolio website operated by Ovilli. All projects showcased are developed by Ovilli. Some pages display third-party advertising served by Google AdSense.
